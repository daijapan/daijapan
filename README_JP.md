## 研究概要

大規模言語モデル（LLM）は急速に社会へ普及しています。しかし、

- AIはどこまで人間を支援できるのか
- その限界は何によって決まるのか
- 数学によってその限界を拡張できるのか

という問いには、まだ十分な理論がありません。

本研究では、**AIによるアクセシビリティを数学として定式化し、その理論を構築すること**を長期的な目標としています。

> **本研究では「アクセシビリティ」を「AIによって人間が到達可能となる能力の集合」と定義します。**
